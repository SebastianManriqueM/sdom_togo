# SDOM Togo - Lome BESS Workshop Materials

This repository contains training and reference material prepared by the
National Laboratory of the Rockies (NLR) on the SDOM model for the Lome BESS
Workshop held in July 2026.

SDOM (Storage Deployment Optimization Model) is an open-source,
Python/Pyomo-based capacity-expansion modeling framework.

- SDOM documentation: <https://natlabrockies.github.io/SDOM/>
- SDOM source repository: <https://github.com/NatLabRockies/SDOM>

## Training Modules

This repository includes the following training modules under `training/`:

1. `training/1_sdom_install/`
	Step-by-step environment setup and SDOM installation (Python, VS Code, uv,
	solver setup, and verification).
2. `training/2_sdom_inputs/`
	SDOM input data structure and data loading with `load_data`, including
	inspection of key tables and active formulations.
3. `training/3_sdom_single_run/`
	End-to-end single SDOM run: model initialization, solve, output export, and
	default plotting.
4. `training/4_sdom_parametric_scalar/`
	Introductory parametric analysis with a scalar sweep (`GenMix_Target`) and
	visualization of case-level trends.
5. `training/5_sdom_parametric_advanced/`
	Advanced multi-dimensional parametric study combining scalar, storage-factor,
	and time-series sweeps.

## Repository Structure

- `data/`: sample and scenario input data used by SDOM.
- `training/`: module-by-module workshop training materials.
- `results/`: location for model outputs generated by runs.
- `references/`: supporting documents and references.
